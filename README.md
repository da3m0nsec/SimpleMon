# SimpleMon

SimpleMon allows you to monitor multiple systems in an easy way, while being customizable, hackable and secure.

## Dependencies

Compiling:
```
sudo apt install cmake g++ libgcc-7-dev libmysqlclient-dev libmysqlcppconn-dev libssl-dev
```

Using Client:
```
sudo apt install openssl
```

Using Server:
```
sudo apt install openssl
```
